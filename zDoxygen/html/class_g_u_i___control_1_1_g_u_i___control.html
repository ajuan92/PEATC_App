Detailed Description

Controlador de la aplicación grafica.

Note
Este módulo despliega la interfaz gráfica que interactúa con el usuario, así como capturar los comandos y datos generados por el usuario, además de implementar la lógica que conecta a la GUI con los módulos de la capa Control_Block
